## 🎯 План улучшений (по приоритетам)

### ПРИОРИТЕТ 1: Schema.org разметка

#### 1.1. Улучшить LocalBusiness schema
- Добавить `priceRange: "€€"`
- Добавить `paymentAccepted: ["Cash", "Credit Card", "Bank Transfer"]`
- Добавить `currenciesAccepted: "EUR"`
- Добавить `hasOfferCatalog` с услугами для района
- Добавить `aggregateRating` (4.9/5, 500+ отзывов)

#### 1.2. Добавить Service schema
Для каждой страницы района добавить несколько Service:
- "Toruabi Lasnamäes" (или Mustamäel/Kesklinnas)
- "Kanalisatsiooni toruabi Lasnamäes"
- "Boileri toruabi Lasnamäes"
- "Avarii toruabi Lasnamäes"

#### 1.3. Добавить Review schema
- 3-5 отзывов для каждого района
- С рейтингом, автором, датой

#### 1.4. Добавить ImageObject schema
- Для главного изображения страницы

### ПРИОРИТЕТ 2: Расширение контента

#### 2.1. Добавить больше H3/H4 заголовков
- Разбить длинные блоки на подзаголовки
- Использовать ключевые слова в заголовках

#### 2.2. Добавить блок "Почему выбирают нас"
- Конкретные цифры: "500+ вызовов в Lasnamäe", "Среднее время прибытия: 35 минут"
- Trust signals: "10+ лет опыта в районе"

#### 2.3. Добавить блок с отзывами
- 2-3 отзыва клиентов из района
- С именем, датой, рейтингом

#### 2.4. Расширить информацию о районе
- Добавить специфику района (панельные дома, новостройки, старые здания)
- Добавить информацию о типичных проблемах в районе

### ПРИОРИТЕТ 3: Внутренняя перелинковка

#### 3.1. Добавить ссылки между страницами районов
- В блоке "Районы обслуживания" добавить ссылки на другие районы
- В футере добавить ссылки на все страницы районов

#### 3.2. Добавить ссылки с других страниц
- На FAQ: "Toruabi Lasnamäes" → ссылка на страницу района
- На блог: в статьях упоминать районы со ссылками
- На hinnakiri: "Цены в Lasnamäe" → ссылка на страницу района

#### 3.3. Добавить контекстные ссылки
- В тексте естественно упоминать другие районы со ссылками
- "Также обслуживаем Mustamäe и Kesklinn"

### ПРИОРИТЕТ 4: Расширение FAQ

#### 4.1. Добавить больше вопросов в FAQPage schema
- Увеличить с 3 до 5-7 вопросов
- Добавить вопросы о конкретных проблемах района

#### 4.2. Добавить вопросы о ценах
- "Сколько стоит toruabi в Lasnamäe?"
- "Есть ли скидки для жителей Lasnamäe?"

### ПРИОРИТЕТ 5: Обновить llm.txt

#### 5.1. Добавить информацию о страницах районов
- Указать новые страницы в структуре
- Описать их назначение
