**Приоритет:** 🔴 КРИТИЧНО

### 2. НЕДОСТАТОЧНО ALT-ТЕГОВ ДЛЯ ИЗОБРАЖЕНИЙ (ВЫСОКО)
**Проблема:** 
- Многие изображения в галерее не имеют alt-тегов
- Некоторые изображения имеют общие alt-теги
- Нет описательных alt-тегов для изображений услуг

**Влияние:** Потеря трафика из Google Images, проблемы с доступностью  
**Приоритет:** 🔴 ВЫСОКИЙ  
**Примеры:**
- Галерея работ - изображения без alt
- Иконки услуг - только aria-hidden="true"

### 3. СЛАБАЯ ВНУТРЕННЯЯ ПЕРЕЛИНКОВКА (ВЫСОКО)
**Проблема:**
- ❌ Нет ссылок между страницами районов (Lasnamäe ↔ Mustamäe ↔ Kesklinn)
- ❌ Нет ссылок на страницы районов с FAQ страницы
- ❌ Нет ссылок на страницы районов с блога
- ❌ Нет ссылок на страницы районов с hinnakiri
- ❌ Мало контекстных ссылок в тексте

**Влияние:** Слабое распределение PageRank, плохая навигация для поисковиков  
**Приоритет:** 🔴 ВЫСОКИЙ

**Текущее состояние:**

- Районы → Главная: ✅ Есть
- Районы ↔ Районы: ❌ НЕТ
- FAQ → Районы: ❌ НЕТ
- Блог → Районы: ❌ НЕТ

### 4. НЕДОСТАТОЧНО H3/H4 ЗАГОЛОВКОВ (СРЕДНИЙ)
**Проблема:** 
- На главной странице только H1 и H2
- На страницах районов мало подзаголовков
- Длинные блоки текста без разбивки

**Влияние:** Хуже структура контента для поисковиков, сложнее читать  
**Приоритет:** 🟡 СРЕДНИЙ

**Примеры:**
- Главная: H1 → H2 → текст (нет H3)
- Районы: H1 → H2 → длинные параграфы без H3
