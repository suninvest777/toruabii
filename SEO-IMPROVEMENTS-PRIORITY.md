# Приоритетные SEO улучшения для 24toruabi.ee

## 🔥 КРИТИЧЕСКИ ВАЖНЫЕ (высокий приоритет)

### 1. **WebSite Schema с SearchAction** ⭐⭐⭐
**Влияние:** Улучшает понимание сайта поисковыми системами, добавляет функцию поиска в Google
**Реализация:** Добавить WebSite schema с SearchAction на главную страницу
**Время:** 15 минут

### 2. **HowTo Schema для статей блога** ⭐⭐⭐
**Влияние:** Статьи могут появляться в Google HowTo rich snippets
**Реализация:** Добавить HowTo schema для статей с инструкциями (например, "как слить воду с радиаторов")
**Время:** 30 минут

### 3. **HTML Sitemap для пользователей** ⭐⭐
**Влияние:** Улучшает навигацию, внутренние ссылки, UX
**Реализация:** Создать страницу /sitemap с ссылками на все важные страницы
**Время:** 20 минут

### 4. **Оптимизация изображений с srcset** ⭐⭐⭐
**Влияние:** Улучшает Core Web Vitals (LCP), скорость загрузки
**Реализация:** Добавить srcset для hero изображений и изображений в статьях
**Время:** 30 минут

### 5. **BreadcrumbList на всех страницах блога** ⭐⭐
**Влияние:** Улучшает навигацию, внутренние ссылки, понимание структуры сайта
**Реализация:** Добавить BreadcrumbList schema на все страницы блога
**Время:** 20 минут

## 📈 ВАЖНЫЕ (средний приоритет)

### 6. **Больше контекстных внутренних ссылок** ⭐⭐
**Влияние:** Улучшает внутреннюю перелинковку, распределение PageRank
**Реализация:** Добавить больше естественных ссылок на страницы районов в тексте статей
**Время:** 1 час

### 7. **Article Schema улучшения** ⭐
**Влияние:** Улучшает отображение статей в поиске
**Реализация:** Добавить article:author, article:published_time, article:modified_time в мета-теги
**Время:** 30 минут

### 8. **Оптимизация шрифтов** ⭐⭐
**Влияние:** Улучшает FCP (First Contentful Paint)
**Реализация:** Добавить font-display: swap, preload для критических шрифтов
**Время:** 15 минут

### 9. **Дополнительные FAQ в FAQPage schema** ⭐
**Влияние:** Больше вопросов = больше возможностей для rich snippets
**Реализация:** Расширить FAQPage schema до 15-20 вопросов
**Время:** 30 минут

### 10. **Структурированные данные для категорий блога** ⭐
**Влияние:** Улучшает понимание структуры блога
**Реализация:** Добавить CollectionPage schema для страниц категорий
**Время:** 20 минут

## 💡 ДОПОЛНИТЕЛЬНЫЕ (низкий приоритет)

### 11. **VideoObject Schema (если есть видео)** ⭐
**Влияние:** Видео могут появляться в поиске
**Реализация:** Добавить VideoObject schema если есть видео на сайте

### 12. **Person Schema для автора блога** ⭐
**Влияние:** Улучшает E-A-T (Expertise, Authoritativeness, Trustworthiness)
**Реализация:** Создать Person schema для автора статей

### 13. **Оптимизация robots.txt** ⭐
**Влияние:** Улучшает индексацию
**Реализация:** Добавить больше директив для разных ботов
